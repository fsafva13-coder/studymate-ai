# StudyMate AI

StudyMate AI is an AI-powered assistant built as part of the DDS AI Application Challenge 2026.

## Problem
Students often struggle to understand complex topics and lack structured study guidance.

## Solution
StudyMate AI provides:
- Simple explanations
- Key learning points
- Practical study tips
- Motivational support

## Tech Stack
- AIRIA (No-Code AI Platform)
- Prompt Engineering
- GitHub (Documentation & Proof-of-Work)

## Current Status (Day 2)
- AIRIA environment set up
- Initial AI workflow created
- Topic explanation feature working
